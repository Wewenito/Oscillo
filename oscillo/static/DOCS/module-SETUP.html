SETUP

The setup section of the oscilloscope is where you will be able to customize the appearance of each signal displayed on-screen and the grid opacity of the screen.

Any changes made in this menu will not be persistent unless the user is logged-in.

Sub-menu overview

The setup sub-menu allows the user to select the color of each signal being displaying in the oscilloscope's screen and the grid opacity.

The user is able to select specific colors depending on the theme of the screen (light or dark).

If the user is looged-in, the styles will be saved for future sessions.

To save the colors and apply the modifications, simply press the 'SAVE' button.

The channel-buttons will also take the colors assigned by the user, this makes it easier to spot which signal is which channel.
