SIZE

This section will focus on the 'SIZE' sub-menu.
The SIZE sub-menu allow the user to resize elements of the page so that they can either fit in a smaller window or allow the screen to take-up more space than usual.

Changing screen size

To change the screen size of the oscilloscope, just open the SIZE sub-menu and select what size you want to apply to the screen.

There is no need to apply the change, the screen size will be adjusted as soon as the user selects it.

⚠ When selecting the 'Maximized' mode, all UI elements will be removed to make as much space possible for the screen. To escape the maximized mode, simply press the escape key. ⚠
